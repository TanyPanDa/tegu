Теги HTML які є на цій сторінці

HTML Мова розмітки гіпертексту (англ. HyperText Markup Language - HTML). Основна мова для опису змісту веб-сторінок. Посилання, параграфи, форми та інші елементи веб-сторінки створюються за допомогою HTML.
  • Center - HTML тег визначає горизонтальне вирівнювання всіх елементів по центру документа.
  • div - Тег використовується, щоб групувати блоки інформації та форматувати її за допомогою CSS.
  • h3 - HTML теґ визначає на сторінці заголовок третього рівня.
  • ul - Маркований (невпорядкований) список в HTML визначається тегом ul (Unordered List Element). Кожен елемент списку повинен починатися з тега li (скорочене від англійського list item - елемент списку).
  • li - Між відкритим і закритим тегами ul повинен бути один або кілька елементів li
  • body - HTML тег призначений для зберігання тіла веб-сторінки (його контенту), що відображається у вікні браузера.
  • head - HTML теґ призначений для зберігання інших елементів, мета яких - допомогти браузеру в роботі з даними.
  • img - Теґ призначений для відображення зображення на Web-сторінці (формати зображень gif, jpeg, png та інші).
  • a - HTML теґ створює посилання на іншу сторіку (Зовнішне посилання) або на певний елемент цієї сторінки(Внутрішне посилання).
  • strong - Тег strong визначає текст, якому надають особливого значення (важливий текст).
  • mark - HTML теґ mark позначає текст як виділений.
  • video - HTML тег video додає, відтворює та керує відеороликом на веб-сторінці.

Теги CSS які існують

CSS Каскадні таблиці стилів (англ. Cascading Style Sheets - CSS). Мова для запису стильового оформлення веб-сторінок. Колір тла, розмір літер та найрізноманітніші візуальні ефекти створюються за допомогою CSS.
  • background - Властивість background задає колір тла, або його зображення.
  • border - Властивість border - універсальна властивість і дозволяє одночасно задати значення таким властивостям як: border-width, border-style, border-color.
  • color - Властивість color задає колір тексту та його оздоблення, котре задане за допомогою властивості text-decoration.
  • width - Властивість width задає ширину елемента.
  • border-color - Встановлює колір всіх кордонів
  • border-style - Властивість border-style дозволяє встановити стиль межі для всіх сторін елементу.
  • border-width - Властивість border-width встановлює товщину для всіх меж елементу.
  • border-radius - Властивість border-radius - універсальна властивість і дозволяє одночасно задати значення округлень всіх кутів.
  • padding - Властивість padding - задає відступ навколо вмісту елемента (тобто, задає відстань між кордоном елемента та уявним прямокутником, котрий обмежує текст).
  • margin - Властивість margin - встановлює зовнішні відступи (поля) навколо елемента. Таким відступом є простір від грані поточного елемента до внутрішнього краю його батьківського елемента.
